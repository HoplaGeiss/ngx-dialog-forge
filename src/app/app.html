ngx-dialog-forge

Declarative Angular dialog library built on the native <dialog> element

@for (t of themes; track t.value) { }

Basic dialog

Close via the X button, the backdrop, or Escape.

<ngx-dialog [(open)]="showDialog" title="Welcome">
  <p>Dialog body content.</p>
</ngx-dialog>

This is a basic dialog with no action buttons. Close button, backdrop close, and Escape close are all active from the global config.

Declarative actions slot

Use ng-template ngxDialogActions with ngxDialogCancel / ngxDialogConfirm directives.

@if (lastAction()) { Last action: {{ lastAction() }} } 
<ngx-dialog [(open)]="showDialog" title="Delete item?" (action)="onAction($event)">
  <p>This action cannot be undone.</p>

  <ng-template ngxDialogActions>
    <button type="button" ngxDialogCancel>Cancel</button>
    <button type="button" ngxDialogConfirm>Delete</button>
  </ng-template>
</ngx-dialog>

This action cannot be undone. The dialog closes automatically when either action button is clicked.

Actions via [actions] input

Shorthand for simple confirm / cancel flows without a template slot.

@if (lastAction()) { Last action: {{ lastAction() }} } 
<ngx-dialog
  [(open)]="showDialog"
  title="Confirm action"
  [actions]="[{ label: 'Cancel', type: 'cancel' }, { label: 'Confirm', type: 'confirm' }]"
  (action)="onAction($event)">
  <p>Dialog body content.</p>
</ngx-dialog>

Pass an array of NgxDialogAction objects to the [actions] input. The (action) output receives the button's type string.

Per-instance overrides

No backdrop close, no Escape, size sm — overrides the global provideDialog() config.

@if (lastDialogClosed()) { Closed: {{ lastDialogClosed() }} } 
<ngx-dialog
  [(open)]="showDialog"
  title="Important notice"
  [closeOnBackdrop]="false"
  [closeOnEscape]="false"
  size="sm">
  <p>Only the X button or action close this.</p>
  <ng-template ngxDialogActions>
    <button type="button" ngxDialogConfirm>I understand</button>
  </ng-template>
</ngx-dialog>

This dialog can only be dismissed by clicking the X button or the action below — backdrop and Escape are disabled.
